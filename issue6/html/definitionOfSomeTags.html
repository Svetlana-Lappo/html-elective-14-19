tag abbr
indicates that the character sequence is an abbreviation.
tag address
is intended to store information about the author and can include any HTML elements like links, text, highlights, etc. It is planned that search engines will analyze the content of this tag to collect information about the authors of the sites.
tag aside
Defines a block on the side of the content for placing headings, links, tags and other information. Such a block is usually called a "sidebar".
tag code
is designed to display one or more lines of text that is program code.
tag caption
is designed to create a header for a table and can only be placed inside a table container
tag cite
marks the text as a citation or footnote to another material.
tag fieldset
is intended for grouping form elements.
tag figcaption
Contains a description for the figure tag
tag legend
is used to create a heading for a group of form elements, which is defined by using the fieldset tag.
tag mark
marks text as selected
tag output
determines the area in which information is displayed, mainly with the help of scripts.
tag optgroup
is a container inside which option tags are located, combined into one group.
tag option
is used to define the list item of the container select
tag picture
serves as a container for one or more source elements and one img element to provide an optimal version of the image for different screen sizes.
tag progress
is used to display the progress of the task completion. Changing the value is done through JavaScript.
tag section
defines the section of documents such as chapters, headers, footers or any other sections. The section tag divides the content into section and subsections.
tag samp
is used to display text that is the output of a computer program or script.
tag summary
indicates a heading for the details tag that can be clicked to expand or collapse information.
tag tfoot
is designed to store one or more rows, which are presented at the bottom of the table.
tag th
is intended to create one cell of the table, which is designated as a main.
tag thead
is designed to store one or more rows that are presented at the top of the table.
tag time
Marks text inside tag as date, time, or both
time track
specifies text tracks for audio or video elements.
tag meter
is used to display a value in a certain known range. Used primarily for displaying numeric values
tag nav
defines site navigation.
tag menu
is intended to display a list of menu items. Similar to ol and ul tags
tag menuitem
defines a command / menu item that the user can invoke from the context menu.
tag keygen
is used to generate a key pair
tag main
is intended to display the main content of the document. The content should be unique and not include typical blocks like site header, footer, navigation, sidebar, search form.
tag embed
is used to load and display objects (for example, video files, flash movies, some sound files, etc.) that the browser does not initially understand.
